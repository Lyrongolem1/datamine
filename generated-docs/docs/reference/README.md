## Quick Reference

### Orbital Regimes

| Regime | Altitude Range | Period | Examples |
|--------|---------------|--------|----------|
| LEO | 200-2,000 km | 90-120 min | ISS, Starlink |
| MEO | 2,000-35,786 km | 2-24 hours | GPS, Galileo |
| GEO | ~35,786 km | 24 hours | Comms satellites |
| HEO | Variable | Variable | Molniya orbits |
